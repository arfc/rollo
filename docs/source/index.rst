================================================
ROLLO (Reactor evOLutionary aLgorithm Optimizer)
================================================

**ROLLO** is a Python package that applies evolutionary algorithm techniques to 
optimize nuclear reactor design. 
**ROLLO** is currently enabled to work with nuclear transport and
thermal-hydraulics software, `OpenMC <https://openmc.org/>`_ and 
`Moltres <https://github.com/arfc/moltres/>`_.
